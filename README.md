# OnlineShop — Интернет-магазин смартфонов 📱

Курсовая работа по дисциплине «Веб-технологии и веб-дизайн».
Приложение представляет собой полнофункциональный интернет-магазин с панелью администратора, корзиной и оформлением заказа.

## 🛠 Технологический стек
* **Платформа:** .NET 8.0
* **Фреймворк:** ASP.NET Core MVC
* **База данных:** SQLite
* **ORM:** Entity Framework Core
* **Фронтенд:** HTML5, CSS3, Bootstrap 5, Razor Views

## ✨ Функционал
### 👤 Для покупателя:
* **Каталог товаров:** Просмотр списка смартфонов с фото и актуальными ценами.
* **Умная корзина:**
    * Добавление товаров (индикация "Уже в корзине").
    * Изменение количества, удаление товаров, полная очистка.
    * Подсчет итоговой суммы.
* **Оформление заказа:** Форма ввода данных (Имя, Телефон, Адрес) с валидацией.
* **Подтверждение:** Страница успешного заказа с номером.

### 👨‍💻 Для администратора:
* **CRUD товаров:** Создание, чтение, редактирование и удаление товаров.
* **Учет остатков:** Управление количеством товара на складе.
